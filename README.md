# MatrixStuds
Tiny 8x8mm ear studs (inspired by Mitxela) with 16 LEDs on each, arranged in a 4x4 matrix. Each matrix is controlled by an Analog Devices MAX32660 microcontroller paired with a Bosch BMA530 accelerometer to react to orientation changes. Each stud features a tiny 3.7v-3.3v step-down converter to accept rechargeable Li-ion cells.
